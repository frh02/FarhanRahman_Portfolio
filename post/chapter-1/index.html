Tutorial I: Clean the DATA

April 9, 2020

Data Science is a culmination of different fields and one of the building blocks of becoming a Data Scientist is Data Analysis.This project deals with the most basic step of Data Analysis - Data Cleaning. Consider this as a walk through to learning R as well as Data Analysis.

For this project, we consider the “dhfr” dataset which is available in RStudio cloud. Although you might have to use the following link dhfr, if you are using RStudio in your PC. Let’s dive into the tutorial.

One of the first steps is to do a background check on the dataset that we will be using.

1. The DHFR Dataset:

There are a few data sets included in caret. One of those data set is DHFR data set. DHFR deals with chemistry problems where the object is to relate the molecular structure of compounds (via molecular descriptors) to some property of interest Clark and Pickett (2000). This data set contains values for 325 compounds. For each compound, 228 molecular descriptors have been calculated. Additionally, each samples is designated as “active” or “inactive”.

The data frame dhfr contains a column called Y with the outcome classification. The remainder of the columns are molecular descriptor values.

1.1. Importing the dataset:

If you do not have RCurl package install you can install the package my using the syntax install.packages("RCurl"). Then we store the data in dhfr data frame. To view the data in R we use the command View(dhfr). The following is the code block which can be implemented in R.

library(RCurl)
dhfr <- read.csv(text = getURL("https://raw.githubusercontent.com/dataprofessor/data/master/dhfr.csv") )
View(dhfr)

This is how the dhfr dataframe should look like:

1.2. Check for any missig Data :

Now, we have to check if there is any data field which is not defined or missing. The easiest way to do this to check through the entire data frame and calculate the sum of those values is zero or not. We can use the following command to check is there is any missing value.

sum(is.na(dhfr))

For our case, this value is Zero, which shows that the data is already curated and without any missing values. So, we corrupt this data by introducing NA values randomly over the rows and columns in our dataset using the following code:

na.gen <- function(data,n) {
  i <- 1
  while (i < n+1) {
    idx1 <- sample(1:nrow(data), 1)
    idx2 <- sample(1:ncol(data), 1)
    data[idx1,idx2] <- NA
    i = i+1
  }
  return(data)
}

Now before introducing NA to the dataset, leave the Y class label (output variable) out, as we don’t want Y to have NA values. This can be done by using:

dhfr <- dhfr[,-1]

Now, we can pass our dataframe through the function which generates NA values randomly, using;

dhfr <- na.gen(n=100,data=dhfr)

Now, we can check for the missing data again using the command sum(is.na(dhfr)), you can see that this value is no longer zero. This means that we have successfully created missing data fields in our data. To better understand you can use the following command in the console to see which rows in our data frame has missing values.

# Lists rows with missing data
missingdata <- dhfr[!complete.cases(dhfr), ]
sum(is.na(missingdata))

You can hover through your data and see that few values are missing.

2. Handling the DATA:

This is the most important part of our data cleaning step, we have two options on how we want to clean the data:

  • Simply omit the data
  • Imputation: Replace missing values with the column’s like mean or median of the data values

2.1. Deleting all the entries:

You can simply delete all entries with missing data using the following command:

clean.data <- na.omit(dhfr)
sum(is.na(clean.data))

2.2 Imputation:

We can further clean the data by replacing the missing values with either the mean of the subsequent row or column values by using the following command:

  • Replace missing values with the column’s MEAN
# MEAN
dhfr.impute <- dhfr

for (i in which(sapply(dhfr.impute, is.numeric))) {
  dhfr.impute[is.na(dhfr.impute[, i]), i] <- mean(dhfr.impute[, i],  na.rm = TRUE)
}
  • Replace missing values with the column’s MEDIAN
# MEDIAN
dhfr.impute <- dhfr

for (i in which(sapply(dhfr.impute, is.numeric))) {
  dhfr.impute[is.na(dhfr.impute[, i]), i] <- median(dhfr.impute[, i],  na.rm = TRUE)
}

You can finally check if there is any missing values in our data set using the sum(is.na(dhfr.impute))
